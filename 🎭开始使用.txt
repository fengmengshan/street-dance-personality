🎭 街舞人格测评系统 - 开始使用

=== 最简单的启动方法 ===

1. 双击 "simple-start.bat" 文件
2. 等待两个服务器窗口打开
3. 在浏览器中访问 http://localhost:3000

=== 如果自动启动失败 ===

使用 "manual-setup.bat" 手动设置：
1. 双击运行，选择选项6（全部执行）
2. 设置完成后，打开两个命令窗口：
   - 窗口1：cd backend 然后 npm run dev
   - 窗口2：cd frontend 然后 npm run dev

=== 系统要求 ===

- Windows 10/11
- Node.js 18或更高版本
- 网络连接（用于下载依赖）

=== 访问地址 ===

前端网站：http://localhost:3000
后端API：http://localhost:8000

=== 功能特色 ===

✅ 62道专业街舞题目
✅ 12种创新人格类型
✅ 科学的心理分析
✅ 美观的街头风格界面
✅ 个性化建议和推荐
✅ 社交分享功能
